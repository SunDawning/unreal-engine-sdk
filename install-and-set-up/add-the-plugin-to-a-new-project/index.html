Add the plugin to a new project

Creating a new project

  1. To launch Unreal Engine, open the Epic Games launcher and click on Library. Choose the correct version (4.27.x) and click Launch. When you launch Unreal Engine (UE4), the Unreal Project Browser automatically opens.

  2. Under New Project Categories, select the development category that best matches your industry.

  3. Select a template for your project, then click Next. The Project Settings page displays. We recommend using a blank template to initialize the plugin.

  4. On the Project Settings page, you can select the quality/performance level, target platform, whether to include Starter Content, and more.

  5. Set the project to a C++ project. This will create a Visual Studio project and fulfills the requirement to have at least one empty C++ class in the source code folder. View the Deployment page for more information.

  6. At the bottom of the Project Settings page, select where you want to store your project, and give your project a name. Click Create Project to finish.

For more information on project settings see Create a New Project from Unreal Engine 4 Documentation.

Install the plugin

The ArcGIS Maps SDK for Unreal Engine plugin is delivered as a zip archive and available on the Esri Early Adopter site.

To install the plugin:

  1. Exit UE4. Then create a Plugins folder in the main project directory.

    Note

    The folder name must be plugins or Plugins.

  2. Extract the ArcGISMapsSDK folder from the .zip file and copy it to the new Plugins folder.

    Note

    If you encounter The path is too long error, extract the .zip first and copy it into the Plugins folder, instead of copying the .zip file to the Plugins folder

  3. Start UE4.
