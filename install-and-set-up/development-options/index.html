Development options

ArcGIS Maps SDK for Unreal Engine enables access to real world maps and 3D contents from the ArcGIS Platform to create interactive 3D GIS applications for Windows, macOS, Android and iOS, and to deploy Augmented and Virtual Reality apps directly to your devices.

Warning

The 4.27 Pre-release plugin doesn't support mobile development. To develop a mobile application, you need ArcGIS Maps SDK for Unreal Engine Beta 2 version instead of ArcGIS Maps SDK for Unreal Engine Pre-release version. To download ArcGIS Maps SDK for Unreal Engine Beta 2 version, go to the Esri Early Adopter site.

The plugin provides three options to use ArcGIS data and other geospatial content within a level:

  • The Details Panel UI provides the option to create a scene without writing any lines of code
  • Blueprints provides the option to create scenes using the node-based visual scripting interface offered in Unreal Engine
  • The C++ API grants access to the plugin source code. This options offers you the most flexibility and customization options

Details Panel UI

The Details Panel UI is the central location to set up your scene without writing a single line of code. It initiates the necessary components and sample code to add ArcGIS functionality to a level.

You can use the Details Panel UI to change between a global or local scene, set the initial position and rotation of the camera and origin, adjust the extent for a local scene, manage layers, as well as change a layer's property during the setup. You can select from a basemap and enable and disable the elevation. If you want to use a custom basemap or elevation source, use the C++ API or Blueprints.

Start Unreal Engine and you should see an ArcGIS Maps button in the Unreal UI toolbar.

Click the ArcGIS Maps button to initialize the Details Panel UI. It will generate an ArcGIS Map Controller Actor. You can find it in the World Outliner panel.

After selecting the ArcGIS Map Controller Actor in the Details panel, you can choose between a global or a local scene, set the initial position of the camera, change the basemap, add data, and organize the layer content.

Tip
  • You can also find the Details Panel UI sample in the Samples folder. To access the folder, click the folder icon in the Content Browser and select ArcGISMapsSDK Content > Samples. If the option ArcGISMapsSDK Content does not appear in the list, click View Options at the bottom right corner of the Content Browser and select Show Plugin Content.

Blueprints

The Blueprint Visual Scripting system in Unreal Engine is a complete visual scripting system based on the concept of using a node-based interface to create gameplay elements from within the Unreal Editor.

There are several methods in which you can create Blueprints, but we recommend using the Level Blueprint.

To open a Level Blueprint for editing, click the Blueprints button in the Level Editor Toolbar and select Open Level Blueprint:

This opens the Level Blueprint in the Blueprint Editor:

There are several methods in which you can place nodes while in Graph Mode. Access the Context Menu by right-clicking in the Graph Editor and searching for ArcGIS. We provide a series of nodes which can be used to create an ArcGISMap.

For more information on Creating Blueprints see Blueprint Basic User Guide.

Tip

  • You can find the Blueprints sample level in the Samples folder. To access the folder, click the folder icon in the Content Browser and select ArcGISMapsSDK Content > Samples. If the option ArcGISMapsSDK Content does not appear in the list, click View Options at the bottom right corner of the Content Browser and select Show Plugin Content.

  • If the node does not appear, uncheck the Context Sensitive checkbox at the top right of the Context Menu.

C++ API

The C++ API allows you to access the plugin source code and allows you to set up custom scenes. You can find classes and components that you can use to set up your scene in the API object model diagram. (The detailed API reference is coming soon.) By using the C++ API, you gain access to additional features such as the ability to change layer visibility during runtime.

To start using the API you can create a new C++ class:

  1. Click the File dropdown menu in the upper-left corner of the editor

  2. Select New C++ Class

  3. The Choose Parent Class menu will display. You can choose an existing class to extend, adding its functionality to your own. Choose Actor, which is the most basic type of object that can be placed in the world, then click Next.

  4. In the Name Your New Actor menu, name your Actor and click Create Class.

    The new script will be created in whichever folder you have selected in the Content Browser and Unreal Engine automatically opens Visual Studio. By default, Unreal Engine uses Visual Studio, but you can select any editor you like.

  5. Add your newly created Actor into the level by manually selecting, dragging, and dropping the Actor from the Content Browser into the Viewport. This will add the Actor to the World Outliner. Click on the Actor in the World Outliner to display its properties in the Details Panel.

Tip

You can find the C++ API sample level in the Samples folder. To access the folder, click the folder icon in the Content Browser and select ArcGISMapsSDK Content > Samples. If the option ArcGISMapsSDK Content does not appear in the list, click View Options at the bottom right corner of the Content Browser and select Show Plugin Content.
