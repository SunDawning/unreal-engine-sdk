Add layers (Details Panel UI)

Learn how to add and manage layers by using the Details Panel UI.

In this tutorial, you'll add raster tile layers, 3D object scene layers, and integrated mesh scene layers from ArcGIS Online in a global scene or local scene. Also, you'll learn how to add local files, how to manage layers such as set visibility, reorder and remove layers.

Prerequisites

Before starting this tutorial, you should:

Follow the UI tutorial in display the globe (UI) or display a specific area (UI).

Steps

  1. In the Add Data panel, select Add From URL.

    Tip

    If you want to add data from a local file, select Add From File button. You'll find Select File button to select a local file.

  2. Type 01 - New York Transit Frequency in the Layer Name field.

  3. Enter the layer's web service URL in the URL field to add a raster tile layer UrbanObservatory_NYC_TransitFrequency.

    Use dark colors for code blocks
     
    1
https://tiles.arcgis.com/tiles/nGt4QxSblgDfeJn9/arcgis/rest/services/UrbanObservatory_NYC_TransitFrequency/MapServer

  4. Select ArcGISImageLayer from the Select Layer Type drop down.

    Tip

    If you want to add a raster tile layer from a tile package, use .tpk or .tpkx file and select ArcGISImageLayer.

  5. Click Add Layer to add the layer to your map.

    Note: In the Layers section you can see the new layer created.

  6. Repeat steps 1-5 to create the other layers.

    • Layer name: 02 - New York Industrial Areas

      ArcGIS Online item: New_York_Industrial

      Use dark colors for code blocks
       
      1
https://tiles.arcgis.com/tiles/nGt4QxSblgDfeJn9/arcgis/rest/services/New_York_Industrial/MapServer

    • Layer name: 03 - New York Population Density

      ArcGIS Online item: NewYorkCity_PopDensity

      Use dark colors for code blocks
       
      1
https://tiles.arcgis.com/tiles/4yjifSiIG17X0gW4/arcgis/rest/services/NewYorkCity_PopDensity/MapServer

  7. Go to Add Data section and select Add From URL again.

  8. Enter the layer's web service URL in the URL field to add a 3D object scene layer New York, USA Buildings, and name the layer 04 - New York Buildings.

    Use dark colors for code blocks
     
    1
https://tiles.arcgis.com/tiles/P3ePLMYs2RVChkJx/arcgis/rest/services/Buildings_NewYork_17/SceneServer

  9. Select ArcGIS3DModelLayer from the Select Layer Type drop down.

    Tip

    If you want to add a 3D object scene layer package and integrated mesh scene layer package, use .slpk file. In case of an integrated mesh scene layer, select ArcGISIntegratedMeshLayer.

  10. Click Add Layer to add the layer to your map.

  11. On the Layers section, change the opacity of the layers 01 and 02 to 90% and 60%. Keep the default opacity value of layers 03 and 04.

    Tip

    You can modify values and strings in the Layers section with presets. Click the drop down arrow next to the layer to see the layer's URL or file path and the layer type.

    Description of the UI icons:
    • Enable / Disable layer visibility
    • Move layer up
    • Move layer down
    • Copy layer URL to clipboard
    • Select a local file
    • Delete layer

You have successfully configured the UI. Click the Play icon and see your map.

Use the WASD Keys to move left/right/forward/backward. Use the left mouse button to pan around the scene, the right button to orbit, and the scroll wheel to zoom in or out.
