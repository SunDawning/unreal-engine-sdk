Add layers (C++ API)

Learn how to add and manage layers by using the C++ API with an example of New York.

In this tutorial, you'll add raster tile layers, 3D object scene layers, and integrated mesh scene layers from ArcGIS Online in a global scene or local scene. Also, you'll learn how to add local files, how to manage layers such as set visibility, reorder and remove layers.

Prerequisites

Before starting this tutorial, you should:

Follow the blueprints tutorial in display the globe (C++ API) or display a specific area (C++ API).

Steps

  1. Open your .cpp file and add necessary headers in the include list.

    Use dark colors for code blocks
        
    1
2
3
4
#include "ArcGISMapsSDK/API/Unreal/Collection.h"
#include "ArcGISMapsSDK/API/GameEngine/Layers/ArcGIS3DModelLayer.h"
#include "ArcGISMapsSDK/API/GameEngine/Layers/ArcGISImageLayer.h"
#include "ArcGISMapsSDK/API/GameEngine/Layers/ArcGISIntegratedMeshLayer.h"
    Tip

    Add headers according to what kind of layer you plan to use:

    • Raster tile layer: ArcGISImageLayer
    • 3D object scene layer/3D object scene layer package: ArcGIS3DModelLayer
    • Integrated mesh scene layer/integrated mesh scene layer package: ArcGISIntegratedMeshLayer

    See the class diagram for more information. (API Reference will be available soon).

  2. In function CreateArcGISMap(), add a raster tile layer (image tile layer) by using the class name ArcGISImageLayer. Get the arcGISMap layer array from arcGISMap by using GetLayers() and add the layer to it. Set the opacity value to 0.9f and the visibility to true.

    Use UrbanObservatory_NYC_TransitFrequency for MyLayer_1.

    Use dark colors for code blocks
      
    1
2
auto layer_1 = Esri::GameEngine::Layers::ArcGISImageLayer("https://tiles.arcgis.com/tiles/nGt4QxSblgDfeJn9/arcgis/rest/services/UrbanObservatory_NYC_TransitFrequency/MapServer", "MyLayer_1", 0.9f, true, apiKey);
arcGISMap->GetLayers().Add(layer_1);
    Note

    When you add layers from a local file, use the file path and format it in the following way:

    Use dark colors for code blocks
     
    1
C:/Users/my_username/Downloads/"YOUR_DATA"

  3. Add another raster tile layer and set the opacity value to 0.6f.

    Use New_York_Industrial for MyLayer_2.

    Use dark colors for code blocks

auto layer_2 = Esri::GameEngine::Layers::ArcGISImageLayer(
    "https://tiles.arcgis.com/tiles/nGt4QxSblgDfeJn9/arcgis/rest/services/New_York_Industrial/MapServer", "MyLayer_2", 0.6f, true,
    apiKey);
arcGISMap->GetLayers().Add(layer_2);

  4. Add one more raster tile layer and set the visibility to false.

    Use NewYorkCity_PopDensity for MyLayer_3.

    Use dark colors for code blocks
       
    1
2
3
auto layer_3 = Esri::GameEngine::Layers::ArcGISImageLayer(
    "https://tiles.arcgis.com/tiles/4yjifSiIG17X0gW4/arcgis/rest/services/NewYorkCity_PopDensity/MapServer", "*MyLayer_3", 1.0f, false, apiKey);
arcGISMap->GetLayers().Add(layer_3);

  5. To add a 3D object scene layer, use the class name ArcGIS3DModelLayer.

    Use New York, USA Buildings for MyLayer_4.

    Use dark colors for code blocks
      
    1
2
auto layer_4 = Esri::GameEngine::Layers::ArcGIS3DModelLayer("https://tiles.arcgis.com/tiles/P3ePLMYs2RVChkJx/arcgis/rest/services/Buildings_NewYork_17/SceneServer", "MyLayer_4", 1.0f, true, apiKey);
arcGISMap->GetLayers().Add(layer_4);

Save and you have successfully configured a scene using the API. You can go back to Unity, click the Play icon, and see your map.

Use the WASD Keys to move left/right/forward/backward. Use the left mouse button to pan around the scene, the right button to orbit, and the scroll wheel to zoom in or out.

You can change the opacity and visibility of added layers (either during runtime or with presets), delete and reorder layers.

To remove a layer:

auto index = arcGISMap->GetLayers().IndexOf(layer_4);
