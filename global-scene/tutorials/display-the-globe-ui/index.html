Display the globe (Details Panel UI)

Learn how to set a basemap and display the globe by using the Details Panel UI.

In this tutorial, you will create a global scene using a Web Map from ArcGIS Online as a basemap. The surface of the scene is defined with an elevation layer, and the camera is positioned to display an area of New York City.

Prerequisites

Before starting this tutorial, you should:

  1. Have an ArcGIS account and an API key to access ArcGIS Online services while you are developing. If you don't have an account, sign up for free.
  2. Ensure your development environment meets the system requirements.
  3. Take Install and setup steps, and create a new project, and select the Details Panel as your development option.

Steps

Create a new level and open the Details Panel UI

  1. Click on File on the Menu Bar and select New Level.

  2. Select Empty Level in the popup window.

  3. Click the ArcGIS Maps button located in the Unreal UI toolbar, it will generate a Map Controller Actor. You can find it in the World Outliner.

  4. In the World Outliner panel select the ArcGISMapController. The UI will appear in the Details panel.

Set up the view mode and camera location

  1. In the World Outliner panel, select the ArcGIS Map Actor. The UI will appear in the Details panel.

  2. Select Global Scene in the View Mode section.

  3. In the World Origin section you can set up the the center of your GIS scene with its coordinates. For the tutorial, set the parameters to these values:

    • Latitude = 40;
    • Longitude = -74;
    • Height = 0;

  4. The Camera Location defines the camera's initial point of view. For the tutorial, set the parameters to these values:

    • Latitude = 40.691242;
    • Longitude = -74.054921;
    • Altitude = 3000;
    • Heading = 65;
    • Pitch = 68;
    • Roll = 0;
    Note

    Altitude uses meters for its units.

Select a basemap and set terrain elevation

  1. On the Base Map section, you can select the base map that best suits your project. For the tutorial select World Imagery on the dropdown menu.

    Preset Basemaps

    By default, the basemap is set to Imagery. You can select a different basemap from the following dropdown list in the Details Panel UI.

    • Imagery
    • Streets
    • Topographic
    • National Geographic
    • Oceans
    • Light Gray Canvas
    • Dark Gray Canvas

  2. Click on Terrain Elevation to create relief in your 3D scene.

    The Details Panel UI uses Terrain 3D as the elevation source.

Set up the sky and the light

  1. Delete everything in the Detail Panel except for what you just added to the scene.

  2. In the Place Actors window, select Lights > Directional Light and drag it into the level to create a Directional Light. For more information about Directional Lights, see Lights.

  3. Select the Directional Light in the World Outliner, and open the Transform section in the Details window.

  4. Set the Rotation to:

    • X: 0
    • Y: -28
    • Z: -28

  5. Set the Mobility to Movable.

  6. In the Light section, change the Intensity Value to 3.1416.

  7. In the Atmosphere and Cloud section, enable Atmosphere Sun Light.

  8. In the Actor > Spawn Collision Handling Method section, select Always Spawn, Ignore Collisions.

  9. In the Place Actors window, select Lights > Sky Light and drag it into the level to create a Sky Light. For more information about Sky Light, see Lights.

  10. Set the Mobility to Movable.

  11. In the Light section, enable Real Time Capture.

  12. In the Place Actors window, select Visual Effects > Sky Atmosphere and drag it into the level to create a Sky. For more information about Sky Atmosphere, see Fog Effects.

  13. In the Planet section, change the Ground Radius to 6378.137207.

Set the API key

An API Key enables access to ArcGIS Online services and resources.

  1. Go to your developer dashboard to get your API key.

  2. Set the API key in the Authentication section.

You have successfully configured configured a global scene with the Details panel UI. Click the Play icon on the Toolbar and see your map.

Use the WASD Keys to move left/right/forward/backward. Use the left mouse button to pan around the scene, the right button to orbit, and the scroll wheel to zoom in or out.
