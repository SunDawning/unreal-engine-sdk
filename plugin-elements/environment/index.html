Environment

Unreal Engine provides many different environment effects to give visual appeal to your ArcGIS scene. Common lighting and visual effect pipelines works with ArcGIS Maps SDK for Unreal Engine.

Use Unreal's New Level templates

ArcGIS Maps for Unreal Engine will work out of the box with Unreal Engine's New Level templates. You can create a Default, TimeOfDay, or Empty level, then click on the ArcGIS Maps toolbar button to add the essential Actors for an ArcGIS scene, or you can create your own custom Actors to initialize an ArcGIS scene.

Lighting and Atmospheric Blueprints

ArcGIS Maps SDK for Unreal Engine comes with several sample Blueprints that assist in accurately positioning Unreal's Sky Atmosphere and Directional Light Actors. The Sample levels each contain an Environment folder in their World Outliners that use these Blueprints.

They can be found in the Plugin Content folder here:

ArcGISMapsSDK Content\Blueprints

Sky Atmosphere

For the detailed information about Unreal Engine's Sky Atmosphere, visit the Unreal Engine Documentation page. You can also learn more about adding ambience and fog in Unreal Engine by visiting the Fog Effects page.

Atmosphere Root Blueprint

The BP_AtmosphereRoot Blueprint Actor rotates a child SkyAtmosphere Actor based on the position and rotation of the ArcGISMapView Actor.

Atmosphere Blending Blueprint

The BP_AtmosphereBlending Blueprint Actor adjusts the height of the atmosphere of a SkyAtmosphere Actor based on the Pawn's distance from the ground.

Directional Light

For the detailed information about Unreal Engine's Directional Light, visit the Unreal Engine Documentation page. You can also learn more about lighting in Unreal Engine by visiting the Lighting the Environment page.

Attach To Active Camera Blueprint

The BP_AttachToActiveCamera Blueprint component is attached to a Directional Light and follows the Pawn. It is set to find and follow the plugin's Sample Default Pawn, but it can be changed to follow any Actor.

Global Shadows Blueprints

The BP_GlobalShadows Blueprint component is attached to a Directional Light. It changes how far you can see the Directional Light's shadows based on the altitude of the ArcGISMapView Actor. Visually, this is used to keep the shadows of 3D objects and terrain elevation visible when the ArcGISMapView is far enough away from the planet.
