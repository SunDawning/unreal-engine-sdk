System requirements

The pre-release plugin supports Unreal Engine 4.27. This page outlines the minimum system requirements you need to run ArcGIS Maps SDK and Unreal Engine 4.27 on Windows, macOS, Android and iOS.

Warning

The 4.27 pre-release plugin doesn't support mobile development. To develop a mobile application, you need ArcGIS Maps SDK for Unreal Engine beta 2 version instead of ArcGIS Maps SDK for Unreal Engine pre-release version. To download ArcGIS Maps SDK for Unreal Engine beta 2 version, go to the Esri Early Adopter site.

ArcGIS Maps SDK system requirements

Actual performance and rendering quality can vary depending on the complexity of your project. For more information on hardware and software system requirements see System requirements for Unreal Engine 4.27 (You can view the requirements for other platforms by selecting it from the drop-down list below the search documentation).
